Everything you need to organize your web

  • Browser sidebar: Attaches to Chrome, Arc, Safari, Brave, or any browser window
  • Workspace organization: Separate bookmark collections with custom-colored workspaces
  • Nested folders: Create hierarchical folder structures with drag-and-drop
  • Local-first: All data stored in a single JSON file on your Mac
  • Inline editing: Rename folders and links directly in the list
  • Search & filter: Quickly find any bookmark across workspaces
  • Auto favicon & title: Automatically fetches website icons and page titles
  • Arc import: Import your pinned tabs and spaces from Arc browser
  • Always-on-top mode: Pin the window to stay visible above all other apps
  • Open source: MIT licensed, built with Swift and AppKit. View source
